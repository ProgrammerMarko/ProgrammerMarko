# Marko Špoljarec

Student računarstva na Veleučilištu u Bjelovaru. Najviše me zanima backend:
Next.js/TypeScript, PostgreSQL i sve što ide uz poslužitelje — Linux, nginx, Docker.

Radim web-aplikacije za klijente: webshop [banknote.hr](https://banknote.hr),
CRM za praćenje skladišta po gradilištima i stranicu s vlastitim CMS-om za
[Hrvatsko numizmatičko društvo](https://www.hrvatskonumizmatickodrustvo.hr).
S profesorom s Veleučilišta radio sam [KPD Agent](https://big-data.hr/kpdagent/app),
alat koji pomoću LLM-a razvrstava unose u KPD šifre (Fiskalizacija 2.0). Sa strane
gradim [igraj-mafiju.com](https://igraj-mafiju.com), online verziju Mafije
(Next.js + Socket.IO server).
